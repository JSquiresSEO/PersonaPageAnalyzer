# PersonaPageAnalyzer
<p>A Chrome extension that analyzes a webpage's text and actions and determines the likelihood a user-provided persona can execute a specified conversion event. A simple way to determine if page layout and page text are removing or adding friction for your target audience. Requires you provide your own Gemini or OpenAI API key.</p>

<p>Available on the <a href="https://chromewebstore.google.com/detail/persona-page-analyzer/hcghpppgbhgllnnafnhmnjijcjkpigeb?authuser=0&hl=en">Chrome Webstore</a></p>
